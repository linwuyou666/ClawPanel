# ClawPanel Pro v5.5.0

发布时间：2026-04-20

## Hermes 平台管理体验升级

- Hermes 平台页对齐 OpenClaw 通道式配置体验，补齐结构化平台表单与统一字段渲染
- 优化 Hermes 侧边栏、概览页与平台详情加载逻辑，减少额外等待并修正启用/状态展示
- 新增面板设置入口，将管理后台密码修改从系统配置中抽离到顶部弹窗

## Lite 数据持久化与升级安全

- Lite 数据目录迁移到安装目录外的持久化路径，避免升级时误删本地面板与 OpenClaw 数据
- Linux / macOS / Windows 安装脚本统一外置数据目录策略，并兼容旧版内置数据目录迁移

## Panel Chat / OpenClaw 兼容与稳定性

- 修复 panel-chat runtime 配置中的旧 schema 字段，兼容新版 OpenClaw
- 增强 panel-chat 群聊增量回显与执行顺序：worker 先回复、summary 后汇总
- 改善任务执行命令解析与 OpenClaw 扩展目录探测，提升运行时兼容性
